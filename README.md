# Currency Converter

A simple and responsive Currency Converter built using HTML, CSS, and JavaScript. The application fetches real-time exchange rates and allows users to convert one currency into another instantly.

## Features

* Real-time currency conversion
* Supports multiple international currencies
* Dynamic currency dropdown menus
* Automatic country flag updates
* User-friendly interface
* Input validation for amount fields
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Currency API
* Flags API

## How It Works

1. Enter the amount you want to convert.
2. Select the source currency.
3. Select the target currency.
4. Click **Get Exchange Rate**.
5. View the converted amount instantly.
